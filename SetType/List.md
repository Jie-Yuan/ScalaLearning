## 拼接列表
 - `::`
 - `++`
 - `List.concat()`
 
```scala
// List of Strings
val fruit = "apples"::("oranges"::("pears"::Nil))

// List of Integers
val nums = 1::(2::(3::(4::Nil)))
```
## 创建统一列表
- `List.fill()`
```scala
val fruit = List.fill(3)("apples") // Repeats apples three times.
val num = List.fill(10)(2)         // Repeats 2, 10 times.
```
## 反向列表顺序
- `List.reverse`
```scala
fruit.reverse
```
